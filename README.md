# 온다스/온다스ETF Kivy 계산기

## 로컬 실행 (PC)
```powershell
pip install kivy
python main.py
```

## GitHub Actions로 APK 만들기 (추천)
1) 이 프로젝트를 GitHub 저장소로 업로드
2) GitHub 저장소에서 Actions 탭 → `Android APK (Buildozer)` 실행
3) 실행 완료 후 `ondas-apk` 아티팩트에서 APK 다운로드

## 참고
- APK 빌드는 GitHub 서버에서 진행됩니다.
- Windows에 WSL2/Ubuntu 설치 없이도 가능합니다.
